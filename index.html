How often should I backup my work?

Make a back up of your current work whenever you stop working. You could use a temporary folder on another drive, in the cloud, or even an email folder for this. Every week consider making a complete back up of your webtech folder to another disk or cloud storage location. If you ever have a problem with your working disk, you can reinstall your Web server, then copy your backup webtech folder to the htdocs folder of your new installation. As you gain some experience, consider working through the Version Control tutorial on the textbook Web site and start using GitHub to back up your work.

How do I work on my coursework files?

Chapter 2 explains this in detail but here are the basic steps to make changes to a file:

  1. Start a text editor
  2. Open the file you want to work on (or create a new file if that's what you want). These files are located in the appropriate chapter folder under your hdtocs/webtech/coursework folder, or in the webtech/samples folder of your installation.
  3. Edit the file as needed
  4. Save the file (use Save to replace the existing file with your new version, or Save As to save the file with a new name, and/or in a new location, if you don't want to lose the previous version of the file).
  5. Now you can open the file in your Web browser to view your changes. To do this either return to this page and click the coursework link above to navigate to the link to your file, or simply type the URL directly in your browser window, for example: http://localhost/webtech/coursework/chapter05/modify1.html

Need to restore an original coursework or samples file?

Sometimes you may make a number of changes to a file in the coursework or samples folders and then wish you could just start again! Your installation includes two folders named COPIES_OF_ORIGINAL_COURSEWORK_FILES and COPIES_OF_ORIGINAL_SAMPLES_FILES. These folders contain duplicates of the original files that came with your installation. Here two different ways to restore a file from these folders:

  1. Use your text editor to open a file from one of these folders. Then use Save As to save the file to the active samples or coursework folder that contains the file you wish to replace.
  2. Use your File Manager to navigate to the folder that contains the original file, then copy this file to the active samples of coursework folder that contains the file you wish to replace. You can then edit the file as needed.

How can I edit this page?

You can edit this file to suite your own purposes. It is located in your webtech folder and named index.html. You may find it useful to add some notes of your own, or you may want to include more links, for example if you create additional folders (see below).

You can also create additional folders if you want to develop your own work. These can be located under the webtech folder or directly under the htdocs folder. Note that files that are located outside the htdocs folder will not be available for viewing on the Web server unless you change the default document directory settings.
